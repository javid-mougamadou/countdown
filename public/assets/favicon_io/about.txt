Favicon généré pour le projet Countdown (ABeeZee via opentype.js → chemin SVG, pas <text> : librsvg/sharp ne rend pas le texte).

- Police : ABeeZee (https://fonts.google.com/specimen/ABeeZee)
- Lettre : C (blanc #ffffff)
- Fond : #1e1b4b (theme-color PWA)

Régénérer les PNG/ICO :
  docker compose exec web npm run generate:favicon

(avec le service web démarré ; sinon : docker compose run --rm web npm run generate:favicon)
